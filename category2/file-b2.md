## SYNOPSIS

Retrieves information about the hardware platform (physical or virtual)

## SYNTAX

```powershell
Get-HardwarePlatform [[-ContinueOnError] <Boolean>] [<CommonParameters>]
```

## DESCRIPTION

Retrieves information about the hardware platform (physical or virtual)

## EXAMPLES

### EXAMPLE 1

```powershell
Get-HardwarePlatform
```
